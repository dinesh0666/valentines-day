## Local Development

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

This will build the project and deploy it to GitHub Pages.
